III. Stative Verbs

In English, if you want to express and state, thoughts, emotions, or senses we can use stative verbs. This verbs like the title said, express states. It is important pay attention!!! this type of verbs cannot used for describe actions, or in continuous form (continuous is a verbal tense, that we have discussed more them (Ing)). We can categorize this verbs Thoughts: Believe, agree, remember, understand, prefer, know.

IV. Present perfect continuous

When we need to express the development of an ACTIVITY that began in the past and progress right now. We can use the present perfect continuous, also we can use this type of verbal tense when we want to express the duration of the activity. With continuous we have use “Ing” at the end of the main verb in the sentence that is known as the present participle form of the verbs.
