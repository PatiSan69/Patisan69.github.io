VIII. Modals verbs

Modals verbs we can use when we need to express ability, possibility, advice or imposibility. This type of verbs are like helping verbs in a sentence because they are always accompanied by the main verb. When we use these type of verb the main verb mainteing their original form: She can go to the pool. My father was able to run 5 milles everyday in the morning.
