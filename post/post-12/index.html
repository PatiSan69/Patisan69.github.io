XII. Comparatives and superlatives.

December 9, 2019

In english when we need compare two things we can use comparatives and If we need to compare more than two we use superlatives.

Comparatives use er and superlatives use est

Example:

  • This university of north of california is better than the university of south.

  • The fruit in this place is more expensive than in the other place.

Rules

When we have one or two syllable word, like this: Spicy, big.

  1. We can add “er” if is a comparatives and if it is a sperlatives we add ¨est¨.

  2. When we have 2 syllable word we can use “more” or “less”.

  3. When we have 3 or more syllable word we can add “more” or “less” plus adjective.
