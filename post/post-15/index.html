XV. Nouns

December 10, 2019

The nouns are the basic buildings blocks of a a sentence. there are person, things, animals, emotions, places, thoughts. In a sentence we could describe a thing and that thing is know as nouns.

With nouns exist a several rules that we need to know when we want to express something in a sentence with correct form. With nouns we have counts and non-counts nouns.

What is this??

Nouns are classsified in counts, this mean that we can count the noun.

Examples with count noun:

  • We have two chairs in this room, so the guests can not sit in this room

  • A thousand of books are in the library.

  • Two cities are in bankruptcy.

In this examples we can see that we use the noun in plural and this is because all of them are counts nouns. Are a few rules when we need a count noun with plural form.

  • When we have one syllabel we can add “S”: Bests

  • When we have a vocal follow with a s we can add “S”: Pencils

  • When we have a vocal follow with a O we can add “S”: Zoos.

  • When we have a consonant and a y we can add “ ies”: Libraries.

  • When we have sh, ch, ss, x we can add “es”: Boxes.

  • When we have consonant and o we can use “es”: Heroes.

With count noun we have a few of them that not follow this rules and they are know like irregular plural noun.

  • Women, children, men, mouse, foot.

All of them have their proper form in plural.

  • Woman, kids, man, mice, feet,

When we use a noncount noun we can not use plural form, so we use singular verbs.

Why?

Because in English grammar we have nouns that not are countable as coffee or sand. Can you imagine count the sand in the beach?. NOOO!! It is impossible.

So, in English We cannot use this type of nouns in plural form.

Wait a minute!!

If I need to express a sentence with a noncount noun. I can not?.

Yes, you can but with something calling quantifiers. Before to talk about quantifiers les’t see an example:

"I want a cup of cofee" correct form

"I want a cofee" Incorrect form.

because a is an article that we use with count nouns also the noun “coffee” is a noncount noun.
