II. Adverbs

December 1, 2019

In the first point, we develop the verbal tense known as present perfect. In this verbal tense frequently used adverbs (just like another verbal tense).

Adverbs are word that accompanies the verbs in sentences. They modify the meaning of the verb.

For the moment I want to put like example the following adverbs:

Ever, Never, Yet, Already

With the adverb Ever we can express an experienced at some time in your life:

  • I has ever kept in the beach for a week.

With this example we can make a idea about the correct use in a sentence of the adverbs.

With Never we can express negative sentence this mean that you not experienced at anytime in your life:

  • I has never ridden in motorcicle, but I will

If we want to express an action not finished but believe you will. We can use the adverb Yet:

  • I have not finished my homework yet.

When you want to express an action that finished and is not need to do it again, we can use the adverb already:

  • She has already brush his hair.
