What is a participle adjective?:

The differences in meaning in "Ing" and "Ed".

When we develop the types of verbal tense and his meaning we say that present participle in a verb is built with Ing and the past participle is build with Ed. This same rule could be followed with adjective. But we need to be careful!

  1. Exciting ; excited

  2. Amusing; amused.

  3. Boring; bored.

These three words (adjective) have the same meaning??

No, have differences but, which are?

When we add “ed” and adjective we make reference at past participle of the adjective and these mean that we talk about the feeling that feels inside us because something causes it and that cause is expressing with add “ing” is an adjective that is known like present participle. In example:

  • When I eat icecream I feel excited.
  • The iceacream is exciting.

But in some cases, we can confunsed when we need to use a participle because like the title said we word with participle and they can be verbs or adjectives, so we need to be careful about the correct use of this participle and pay attention and how we use this words.

  • They are laughing.

In this sentence, laughing is a verb or and adjective?.

How we can know this?

Well, this is tricky point and in the beginning is a little confused but exist a proof that is known like “the very test”.

The very test consist in add before the participle and adverb and that adverb is “very”. if we add this adverb and the sentence look like make sense in this case we have and adjective, in the oposite case is a verb

So, In the sentence, “laughing” is?

Is an verb.
