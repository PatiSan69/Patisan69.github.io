V. Present perfect VS Present perfect progressive

December 2, 2019

In the last topic we develop the verbal tense know as present perfect continuous but for the student could be confused the differences and the correct use between the present perfect (is the first topic explained)and present perfect continuous. The reason for this is we can to express and action or event with present perfect and present perfect continuous and in the two forms is correct but because of that it is important to know the differences and similarities between this verbal tense.

Similatiries between present continuous and present perfect continuos

  1. We can use present perfect and present perfect continuous for expresing habitual action
  2. Both starts in the past and has some importance in the present time
  3. Uses since and for

Differences between present continuos and present perfect continuous

Present Perfect

  1. With present perfect we can put emphasis on the result of an action.
  2. We can use non-action verbs, such as: Stative verbs.
  3. We can use this verbal tense with the adverbs ever and never

Present perfect progressive

  1. In present perfect progressive we talk about ongoing actions
  2. Emphasis in duration of the action
  3. When the action is finished but you can see the results.
